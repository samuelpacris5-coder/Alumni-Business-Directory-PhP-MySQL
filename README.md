## 📌 Description
The Alumni Business Directory System is a full-stack web application built with PHP and MySQL that allows alumni to showcase and promote their businesses. It provides a centralized platform where users can browse, search, and connect with alumni-owned businesses.

---

## 🚀 Features
- 👤 User registration & login system  
- 🏢 Business listing & management  
- 🔍 Search and filter businesses  
- 📂 Categories for different industries  
- 📱 Fully responsive design  
- 🔒 Secure backend (prepared statements & validation)  

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** Apache (XAMPP/WAMP)
